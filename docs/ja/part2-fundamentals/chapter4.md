# 第4章 Pythonとエンジニアリング基礎

## 本章で学べること

「実践的な」基礎篇へようこそ！

来这里之前、你可能看过一些Python教程——变量、数据类型、循环、関数、面向「零基础小白」那种。教える内容没有问题，但是否有种感觉：学完了还是不知道这玩意儿跟AI开发有什么关系？

これが私が解决しようとする核心的な問題です。

**この章はPython入門ではなく、Pythonの「AI開発スキルギャップ弥补」です。** 変数が何であるかは教えません（知っているはず）、`if-else`の書き方も教えません（見たことがあるはず）。教えるのは：公司でAIに入ると第一天打开代码编辑器，那些让你头皮发麻的东西到底是个啥。

读完这章你会带走：

- **dataclass/pydantic**使用场景——为什么AI代码里满屏都是这玩意儿
- **asyncio异步编程**の实战套路——批量API呼び出し必须有技能
- **类型注解**的正确用法——让Copilot真正帮到你
- **健壮的HTTPクライアント封装**——大模型API调用のストリーミング、リトライ、タイムアウト
- **Gitワークフロー**的正确姿势——monorepo、pre-commit、.env安全
- **2026年の環境管理方案**——uv凭什么这么火
- **GPUサーバーサバイバルスキル**——查日志、看进程、调资源不求人
