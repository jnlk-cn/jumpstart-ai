# 第5章 LLM呼び出しとプロンプトエンジニアリング

## 本章で学べること

おめでとうございます！やっと本当のことができます！

前面几章我们聊了不少「虚」的内容——行业认知、入门路线、工程基礎。この章から、AIアプリ開発の**コア技術領域**に足を踏み入れます。

この章は本の最もコアな技術章の一つです。原来的内容讲了些基础技巧，说实话有点浅——就像是教你「筷子の使い方」を教えるが、「滚动する肉丸子怎么夹」までは教えてくれない那种。我会给你们补上这些实战中真正有用的东西。

读完这章，你能带走：

- **磐石一样的API呼び出し**：调通就行不够，还要能扛住高并发、处理流式输出、在模型翻车时优雅降级
- **系统的なプロンプト方法論**：不是背技巧，而是理解为什么这样写有效，以及什么时候该用什么技巧
- **本番급構造化出力**：JSON Schema怎么做、多级嵌套怎么搞、信息抽出pipeline怎么搭
- **贤いコンテキスト管理**：对话状态机、意図识别、スロット填充——让AI真正「懂」用户
- **本当のコスト管理**：不是理论，是一个月烧了多少钱、怎么砍到一半的经验
- **坑踩の血涙史**：踩过的坑、不想让你再踩一遍
