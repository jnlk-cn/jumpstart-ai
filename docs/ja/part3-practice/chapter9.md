# 第9章 LangChain実践

## 本章で学べること

みんな实战の時間だ！

前面我们打好了基础——会调用大模型了（第5章）、知道怎么给AI接知识库了（第6章）、也搞明白了Agent是怎么工作的（第7章）。现在是时候把这些能力**串起来变成真正能用的もの**了。

この章ではLangChainを解説します——現在最も人気のAIアプリ開発フレームワーク。

正直に言います：**LangChainは入门简单、用好难**。公式のチュートリアルはHello World级别なのに、本番環境に投入すると思っていたより様々な问题が起きてしまう：バージョン非兼容、デバッグ難しい、パフォーマンスが悪い等等...

でも！LangChain依然是目前最主流的AI应用开发框架，学会它找工作加分、项目効率翻倍。而且2025-2026年、LangChainは**LangGraph**を发表、这东西直接把复杂Agent编排变成了声明式编程——就像从汇编进化到Python一样爽。

因此この章では：

- LangChainアーキテクチャとコアモジュールの快速チェック
- LCELの深掘り——LangChainの真髄，必须搞透
- **見せ場：LangGraph**——2026年Agent開発標準ツール，我给你们掰开了揉碎了讲
- RAG实现をトイから本番级へ升级
- マルチツールAgentの深掘り
