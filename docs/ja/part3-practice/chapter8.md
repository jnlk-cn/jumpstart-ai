# 第8章 プロンプトエンジニアリングからエージェント開発へ

## 本章で学べること

正式开始之前、先に一つ質問させてください：假设你是2023年3月入行的AI开发者，那时候最火的是什么？是Prompt工程师——听起来很酷但实际上就是「会写提示词」的岗位。当時のAIアプリ開発は简单得令人发指：你写一个Prompt，AI返回答案，齐活。

但是如果现在（2026年）还想靠「写Prompt」这招走天下，那大概率会饿死在沙滩上。

この章の核心任务是帮你**建立全局视野**。你会清楚地看到：

1. **为什么Prompt工程只是起点**：它能做什么，它的天花板在哪里
2. **Chain是怎么解决这些天花板的**：最早的工程化实践是什么样的
3. **Agent为什么是必然演进**：从「我让你做什么」到「你自己决定做什么」
4. **Agent助理意味着什么**：加上记忆、人格、长期目标后的质变
5. **你现在应该站在哪里**：2026年のエコシステムマップと選定基準

**前置知识**：第5章プロンプトエンジニアリング基礎、第7章AIエージェントコアロジック。
