# 第11章 エージェント開発実践とSkill開発

## 本章で学べること

- エージェント助理と一般的なエージェントの本质的な区别を理解する
- メモリシステムの3層アーキテクチャをマスター：短期メモリ、長期メモリ、ワークメモリ
- 設計原則、インターフェース仕様、エラー処理を含む完全なSkillの開発方法を学ぶ
- Skill注册发现メカニズムと组合ワークフローの設計方法を理解する
- Coze、Dify、LangGraph 3つのフレームワークの適用シナリオを比較する
- 実行可能なAI開発助理の完全なコードを取得する
- エージェント助理の評価体系和商業化路径を理解する

2024年是Agent元年。でも另外一件事我也想说：2024年也是Agent助理产品经理噩梦的开始。

因为「Agent」这三个字被用烂了。什么都敢叫Agent，什么都不是真正的Agent。因此この章では、何が本当にAgent助理なのかを明らかにし它和普通Agent的区别在哪里，为什么这种区别重要。然后、設計からコードまで、本当に動くAgent助理を作成します。

特に重点的に解説するのはSkill開発です——这是让Agent从「能说会道」变成「能干活」の键となる能力ユニット。
