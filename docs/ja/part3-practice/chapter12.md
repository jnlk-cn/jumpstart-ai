# 第12章 サービスデプロイとエンジニアリング

## 本章で学べること

ローカルマシンで動くAIアプリケーションを構築することは別として、何千ものユーザーに対して24時間365日安定して動作させることは全く別の問題です。

この章の内容：

- **サービスデプロイの必須知識**：Docker、Kubernetes、クラウドプラットフォーム
- **APIデザインのベストプラクティス**：REST、認証、速率制限
- **モニタリングと可观测性**：ログ、メトリクス、アラート
- **AIアプリケーションのCI/CD**：自動テストとデプロイメントパイプライン
- **コスト最適化**：APIコスト管理、キャッシング戦略
- **セキュリティ考量事項**：APIキー、データプライバシー、プロンプトインジェクション防御
