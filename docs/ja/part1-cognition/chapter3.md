# 第3章 あなたの 入行 ロードマップ

## 本章で学べること

前2章では「能不能入行」と「要不要现在入行」の問題を解決しました。この章は認知篇の締めくくりとして、あなたに**そのまま実行できるロードマップ**を示します。

读完这章你会知道：

- あなたはどんなタイプの 入行 者か（後端/前端/ゼロベース/学生）、そしてあなただけの戦略
- 8週間 入行 スプリントプラン、毎週何をすべきか精确
- 6ヶ月深掘り進化プラン、「能动く」から「站得住」へ
- 学習リソースの選び方，哪些值得花钱，哪些免费够用

看完这章，你应该就可以关掉书，打开电脑，开始执行了。

---

## 3.1 四タイプの入門者像とそれぞれの戦略

ロードマップを描く前に、首先问你一个问题：**你是谁？**

AIへの 入行 に标准答案はない。出発点が異なれば、最適なパス也不同。

入門者を4つのタイプに分け、各タイプ都有自己的优势和坑。对号入座，找准属于你的策略。

---

### タイプ1：後端からAI開発への轉身

**这样的人有哪些特徵**

1〜5年の後端開発経験があり、Java、Go、Python都可能、APIを書いたりサービスを構築したりデータベースと向き合ってきた，對你来说写代码不是问题，问题是「AI那边的代码怎么写」。

**你的优势**

最大の优势は**エンジニアリング能力**。很多人以为转AI要放弃工程能力，其实恰恰相反——2026年最も不足している人材は、AIモデルを安定して本番で動かせる人。

你懂高并发、服务治理、API设计、部署运维。这些东西、纯AI背景的人反而是短板。你的工程经验是你转型的护城河，不是负担。
