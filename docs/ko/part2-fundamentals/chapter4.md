# 제4장 Python과 엔지니어링 기초

##本章에서 배울 것들

"진짜 실전" 기초편에 오신 것을 환영합니다!

来到这里之前,你可能看过一些Python教程——变量、数据类型、循环、関数、"零基础小白"용. 내용은 문제없지만,这种感觉是否 있습니까: 学完之后还是不知道这跟AI开发有什么关系?

이것이 제가 해결하려는 핵심 문제입니다.

**이 챕터는 Python 입문이 아니라 Python의 "AI 개발 크래시 코스"입니다.** 변수가 무엇인지教えない(你已经知道), `if-else` 작성方法도教えない(你已经见过). 무엇을 가르칠까요: AI 회사에서第一天上班时打开代码编辑器, 那些让你头皮发麻的东西到底是什么.

이 챕터를 읽고 나면 가져갈 것들:

- **dataclass/pydantic** 사용 시나리오 - 왜 AI 코드에 이것들로 가득 찬지
- **asyncio** 비동기 프로그래밍 실전 패턴 - 배치 API 호출의 필수 기술
- **타입 힌트**의 올바른用法 - Copilot이 정말 도와주게 하는 방법
- **견고한 HTTP 클라이언트** 래핑 - LLM API 호출의 스트리밍, 재시도, 타임아웃
- **Git 워크플로우**의 올바른 자세 - monorepo, pre-commit, .env 보안
- **2026년 환경 관리 솔루션** - uv가 왜 이렇게 핫한지
- **GPU 서버 생존 기술** - 로그 확인, 프로세스 확인, 리소스 디버깅
