# 제12장 서비스 배포와 엔지니어링

##本章에서 배울 것들

로컬 머신에서 작동하는 AI 애플리케이션을 구축하는 것은 한 가지이지만, 수천 명의 사용자에게 24시간 안정적으로 작동시키는 것은 완전히 다른 문제입니다.

이 챕터의 내용:

- **서비스 배포 필수 지식**: Docker, Kubernetes, 클라우드 플랫폼
- **API 설계 모범 사례**: REST, 인증, 속도 제한
- **모니터링과 可观测性**: 로깅, 메트릭스, 알림
- **AI 애플리케이션 CI/CD**: 자동화된 테스트와 배포 파이프라인
- **비용 최적화**: API 비용 관리, 캐싱 전략
- **보안 고려 사항**: API 키, 데이터 프라이버시, 프롬프트 인젝션 방어
