# Chapter 5: LLM API & Prompt Engineering

## What You'll Learn

Congratulations! You're finally getting to the real stuff!

In previous chapters, we talked about a lot of "abstract" things—industry cognition, entry roadmap, engineering basics. Starting from this chapter, we're entering the ** hardcore area** of AI application development.

This chapter is one of the most core technical chapters in the book. The original version taught some basic techniques, but honestly, it was a bit shallow—like teaching you "how to use chopsticks" but not telling you "how to pick up rolling meatballs." I'll supplement these truly useful things from real practice.

After reading this chapter, you will take away:

- **Rock-solid API calls**: Not just making them work, but handling high concurrency, streaming output, and graceful degradation when models fail
- **Systematic Prompt methodology**: Not memorizing tricks, but understanding why certain approaches work and when to use which technique
- **Production-grade structured output**: JSON Schema, multi-level nesting, information extraction pipelines
- **Smart context management**: Conversation state machines, intent recognition, slot filling
